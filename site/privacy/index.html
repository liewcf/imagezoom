Image Zoom Privacy Policy

Last updated: 2026-05-29

Image Zoom does not collect, store, sell, or share user data.

The extension runs locally in your browser. It looks at image elements on the web page you are viewing so it can zoom selected images inline or show them in a full-page overlay.

The extension does not use analytics, cookies, tracking, accounts, remote code, or a server controlled by the developer.

When a page provides larger image candidates through srcset, Image Zoom may ask the browser to load one of those existing page image URLs for the overlay. This is only used to show a sharper zoomed image. No data is sent to the extension developer. The website hosting the image may receive the normal image request.

To remove the extension, uninstall it from chrome://extensions. This removes the extension and its local behavior.
